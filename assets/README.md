# Assets Directory

This directory contains static assets for the mobile application.

## Expected Files

- `icon.png` - Application icon (1024x1024 recommended)
- `splash.png` - Splash screen image (2048x2048 recommended)
- `adaptive-icon.png` - Android adaptive icon (512x512 recommended)
